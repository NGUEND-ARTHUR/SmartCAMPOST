# Playwright E2E tests (scaffold)

This folder contains Playwright E2E test scaffolding generated by the assistant. Tests are TypeScript, use reusable API helpers, and are organized by workflow.

Environment variables supported (optional):

- `E2E_BASE_URL` — base URL for the app (defaults to <http://localhost:8080>)
- `TEST_ADMIN_EMAIL`, `TEST_ADMIN_PASSWORD` — admin credentials for API tests
- `GOOGLE_TEST_IDTOKEN` — optional Google ID token for integration test
- Role-specific test credentials JSON may be provided via `TEST_FINANCE_CREDENTIALS`, etc.

Notes:

- Tests include API-level checks (using Playwright `request`) and basic UI checks where applicable.
- Tests do not execute anything automatically — run with `npx playwright test --config=playwright-tests/playwright.config.ts` when ready.
