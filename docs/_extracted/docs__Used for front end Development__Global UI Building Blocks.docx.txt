Global UI Building Blocks (Used Everywhere)
API + State hooks (recommended)
useAuth() → token, role, user profile, logout
useRoleGuard(requiredRoles[])
useApiClient() → axios/fetch wrapper with Authorization header + error handling
useQuery / useMutation wrappers (React Query style):
useFetchList(key, url, params)
useFetchOne(key, url)
useCreate(url)
useUpdate(url)
useDelete(url)
Global UI states (all pages)
idle, loading, success, error
empty
forbidden (403)
notFound (404)
conflict (409) (unique constraints like pickup per parcel, unique phone/email)
frozenAccount (user_account.frozen → block actions)
Common components
AppLayout(role) + Sidebar(role) + Topbar
PageHeader(title, breadcrumbs, actions)
KpiCards
DataTable (search, filters, pagination)
StatusBadge (ParcelStatus, PaymentStatus, PickupState, TicketStatus, RefundStatus, RiskStatus)
ConfirmModal (danger actions)
Toast + ErrorBanner
EmptyState + SkeletonLoader
Map components
MapView
AddressPickerMap (pin + search)
RoutePreviewMap (polyline + distance/duration)
AgencyMarkersMap
ReadOnlyTrackingMap
