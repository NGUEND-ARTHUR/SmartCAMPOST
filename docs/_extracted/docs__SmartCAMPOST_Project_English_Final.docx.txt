Chapter 1: Introduction
💡 1. Final Project TitleSmartCAMPOST: Intelligent Parcel Tracking, Collection, and Delivery System Using QR Code, GPS, and Digital Payments.Why this name is perfect:- It keeps “Smart” to reflect innovation and modernity.- It integrates CAMPOST to show institutional context.- It emphasizes the three axes: Tracking, Collection, and Delivery.- It mentions the main technologies: QR Code, GPS, and Digital Payment.- It is globally understandable and academically professional.🧩 2. Project DescriptionSmartCAMPOST is an intelligent and integrated application that allows CAMPOST clients to send, track, and receive their parcels digitally and transparently.Through a system of QR Codes, GPS geolocation, and automatic SMS/app notifications, every parcel is tracked in real time from dispatch to delivery.Clients can:- Drop off parcels at a CAMPOST branch or request home collection via the app.- Track their parcels at every step (in transit, arrived, delivered).- Choose between branch pickup or home delivery.- Make payments directly within the app (via Orange Money, MTN Money, or CAMPOST Money).The system also includes an administrator dashboard for CAMPOST to:- View all parcels in the network, their status, and history.- Generate reports and invoices.- Supervise agents and optimize logistics.This project aims to modernize Cameroon’s postal service, reduce queues, and improve customer satisfaction through full digitalization of the logistics process.
Chapter 2: System Workflow and Analysis
🧩 1. When and How the QR Code is Generated📦 Step: Parcel RegistrationWhen a client wishes to send a parcel, there are two options:Option A — Drop-off at CAMPOST Branch:- The client fills a digital or paper form (weight, content, value, destination, etc.).- The agent registers the parcel in the application.- The system automatically generates a unique ID (e.g., CAM-2025-000123).- From this ID, a QR Code is generated.- The QR Code is printed on a small sticker and attached to the parcel.- The client receives an SMS and a digital receipt with the tracking number.Option B — Home Collection Request (via the App):- The client submits a pickup request with parcel description (weight, type, fragility, etc.).- The app pre-registers the parcel and generates a temporary QR code.- When the agent arrives, they scan the QR code using their mobile app to confirm pickup.- If not printed, the agent can print a label using a Bluetooth printer or attach a pre-coded postal sticker.💡 Why print the QR Code?- The QR code serves as a unique identifier throughout transport.- It links to all parcel data (sender, receiver, description, payment, history).- Each scan updates the parcel’s status in real time.- It ensures fast, reliable tracking with no manual entry.👉 Yes, the QR Code must be printed on a label (with CAMPOST name, parcel ID, and optional barcode).🧾 2. Parcel Description and Agent ValidationDuring registration or pickup, the client describes their parcel:- Type of content (documents, clothes, electronics, etc.)- Estimated weight- Declared value- Fragility (checkbox “Fragile”)- Destination- Option: photo of the parcel before sealingWhen the agent receives the parcel:- They verify the client’s description.- If correct, they click “Validate Description.”- If discrepancies exist, they adjust details, add comments (e.g., “Weight corrected: 5.2 kg”), and attach a photo.- Once validated, the system updates the parcel status to “Registered and validated by agent,” triggering an SMS to the client.
