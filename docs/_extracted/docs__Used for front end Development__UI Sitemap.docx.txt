10) Recommended “Menu Locking Rules” (Permissions)
If role ≠ FINANCE, hide Refunds
If role ≠ RISK, hide Risk Alerts / Compliance
If role ≠ ADMIN, hide Master data CRUD (agencies, staff, agents, couriers, tariffs, integrations, ussd)
AGENT sees only operational pages (counter + scans)
COURIER sees only pickups/deliveries/route
CLIENT sees only client self-service
